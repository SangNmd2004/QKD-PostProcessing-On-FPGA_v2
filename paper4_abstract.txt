Large-Scale and High-Speed FPGA-Based Privacy
Ampliﬁcation for Quantum Key Distribution
Bing-Ze Yan , Qiong Li , Hao-Kun Mao , Hong-Wei Xu, and Ahmed A. Abd El-Latif
Abstract —In this paper, a high-performance ﬁeld-
programmable gate array (FPGA) based privacy ampliﬁcation
(PA) scheme for both continuous variable and discrete variable -
quantum key distribution (QKD) is presented, which can provide
more than 1.5 Gbps throughput at the input block size close to
108. Besides, the resource consumption is so economical that there
is no need of any external storage. To the best of our knowledge,
our scheme outperforms existing FPGA based PA schemes by
over an order of magnitude in terms of throughput and input
block size. To achieve this feature, we design a novel FPGA-based
hybrid hash PA algorithm combining multilinear modular hash
and modular arithmetic hash (MMH-MH) based on number
theoretical transform (NTT). Based on the hybrid hash algorithm,
many optimization strategies are designed in our FPGA based PA
scheme, such as an NTT-based optimization unit, an NTT-reusable
architecture, a memory-saving modular accumulation unit
for multilinear modular hash and a pipelined binary modulo
unit for modular arithmetic hash. Plentiful experiment results
demonstrate the high performance and reliability of our scheme.
It can be embedded in most standalone QKD devices conveniently.
What’s more, it is also a good candidate for a chip-based QKD
system.
Index Terms —FPGA, multilinear modular hash, number
theoretical transform, privacy ampliﬁcation, quantum key
distribution.
I. I NTRODUCTION
QUANTUM key distribution (QKD) is a notable technique
which exploits the principle of quantum mechanics to
perform the information theoretical secure key distribution be-
tween two remote parties, named Alice and Bob [1]. QKD can
be divided into discrete variable (DV) QKD and continuous
variable (CV) QKD. A QKD system can be divided into two
parts, the quantum optical subsystem and the post-processing
subsystem. The quantum optical subsystem is in charge of

The post-processing subsystem mainly conducts error correction
and privacy ampliﬁcation [2].
Privacy ampliﬁcation is a necessary part of the post-
processing subsystem in quantum key distribution [3]. It is an
art of distilling a highly secure key from a partially secure string
by public discussion between two parties. PA requires high
throughput to ensure timely processing of input key, and the
input block size of PA has signiﬁcant impact on the ﬁnal key rate
of QKD system due to the ﬁnite-size effect [4], [5]. Therefore,
throughput and input block size are two key indicators of PA.
Field-programmable gate array (FPGA) is a popular platform
to design QKD systems, including PA. It has a high integration
density and can handle both the system control task and the cal-
culation task on one chip. Furthermore, the power consumption
of a FPGA is much lower than that of other platforms. However,
low throughput and small input block size of PA are still the
main bottleneck of the FPGA-based QKD system.
Several FPGA-based post-processing subsystems and PA
schemes have been designed for high-performance QKD sys-
tems [6], [7], [8], [9]. The small input block size is the most
critical problem of FPGA-based PA schemes. The largest input
block size of existing FPGA-based PA schemes is 106[8],
[9], while 106is only the minimum requirement of a common
QKD system. The input block size is expected to be much
larger than 106to realize a negligible ﬁnite size effect on the
secure key rate in a high-performance QKD system [10]. The
throughput of FPGA-based PA schemes is also insufﬁcient for
high-performance QKD system. The highest throughputs of
existing FPGA-based PA schemes are 116 Mbps with the block
size of106[9] and 186 Mbps with the block size of 349,952 [8].
However, as the repetition frequency of QKD systems increases
and wavelength division multiplexing technology develops, the
raw key rate of a QKD system may exceed several gigabits per
second and the throughput of a PA scheme is required to the
same order of magnitude [11].
In this paper, a large-scale and high-speed (LSHS) FPGA-
based privacy ampliﬁcation (PA) scheme is presented. Its
throughput exceeds 1.5 Gbps and its input block size is close
to108. An FPGA-based hybrid hash PA algorithm combining
multilinear modular hash and modular arithmetic hash (MMH-
MH) is designed for our scheme [12]. This algorithm achieves
the best computation complexity O(nlogn)with the optimiza-
tion method based on number theoretic transform (NTT) [13].
Besides, an NTT-reusable structure and its control unit for
our scheme is designed to reduce the resource consumption.

II. MMH-MH PA A LGORITHM
An FPGA-based MMH-MH PA algorithm is designed in this
section. It is the foundation of the LSHS PA scheme.
The main process of a typical PA algorithm is compressing
the input sequence with a hash function randomly chosen from
a universal hash family [14]. both the multilinear modular hash
and the modular arithmetic hash are universal hash families.
The multilinear modular hash can not only split the input key
and handle it separately but also compress the input key from a
binary sequence with length nto a2B-nary sequence with length
n/B . The computation complexity of multilinear modular hash
can be reduced to O(nlogn)by NTT-based optimization [15].
Therefore, the calculation amount of multilinear modular hash
is much lower than other universal hash family. However, the
output set of the multilinear modular hash is a ﬁxed prime
ﬁeld, which is unﬁt for PA. Therefore, we designed a new
PA algorithm combining the multilinear modular hash and the
modular arithmetic hash, which utilizes the variable length bit
output set of the modular arithmetic hash.

with different universal hash families instead of once. The main
